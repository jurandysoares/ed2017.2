# Laços

## While

```
1. Iniciar variável de controle (VC)
2. Enquanto (teste com VC) faça
  início
    3. Bloco do enquanto
    4. Atualização da VC
  fim
```

Exemplo 1 em Python:
```python
i = 1
while i<=100:
  print(i, end=' ')
  i += 1
```
Exemplo 2 em Python:
```python
k = int(input('Limite: '))
i = 2
while i<= k:
  print(i, end=' ')
  i += 2
```
